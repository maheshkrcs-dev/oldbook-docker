# 📚 OldBook - Dockerized Web Application

## 🚀 Project Overview

OldBook is a PHP + MySQL based web application that allows users to buy and sell old books.

This project is fully containerized using Docker for easy deployment and migration across systems.

---

## 🛠️ Tech Stack

* PHP (Apache)
* MySQL
* phpMyAdmin
* Docker & Docker Compose

---

## ⚙️ Features

* User Registration & Login
* Book Listing & Selling
* Database-driven application
* Fully Dockerized setup

---

## 🐳 Run Project (1 Command Setup)

```bash
git clone https://github.com/maheshkrcs-dev/oldbook-docker.git
cd oldbook-docker
docker compose up -d --build
```

---

## 🌐 Access

* Website → http://localhost:8080
* phpMyAdmin → http://localhost:8081

---

## 🗄️ Database

* Auto-import using Docker init scripts
* No manual setup required
